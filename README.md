# Dan's Fork of DWM 
> My own config and styling for the Dynamic Window Manager

## Requirements
Like good old DWM, you'll need the Xlib header files.

## Installation
- First clone the repo and edit config.mk to your liking.
- Then, you can run `make clean install` as root to install

## Usage
I recommend making your own .desktop file and adding it to /usr/share/xsessions/, 
so you can choose between DWM and your *normie* desktop environment,
but you can add `exec dwm` to .xinitrc like a madlad either way
